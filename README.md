## Xianyipin Snack Sales Trading Platform

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/776eb30d86034de08443b69b1bd5f617/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular Users, Administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Change Password
4. Personal Information
5. Order Review Management
6. My Favorites Management
7. Order Management
8. Snack Information
9. Snack Information
10. My Address
11. Shopping Cart
12. Online Customer Service


**Administrator Functions:**
1. Login
2. Personal Center (Edit Information, Password)
3. User Management
4. Snack Category Management
5. Snack Information Management
6. Order Review Management
7. Banner Management
8. Snack News
9. Online Customer Service
10. Order Management
